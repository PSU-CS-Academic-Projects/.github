# Contributing to PSU-CS-Academic-Projects

Thank you for contributing to PSU-CS-Academic-Projects.

Before submitting a project, please ensure that your repository follows the organization standards.

## Submission Requirements

Each repository must include:

* A clear and descriptive repository name
* A completed README.md file
* Project description
* Feature list
* Installation guide
* Screenshots
* Team member information
* Academic year
* Subject information
* GitHub topics

## Recommended Information

The following information is strongly recommended:

* Live demo link
* Documentation
* Video demonstration
* Presentation slides
* Future improvements

## Repository Naming

Use clear project names.

Examples:

* Student-Attendance-System
* Library-Management-System
* Campus-Navigation-App
* Online-Enrollment-System

Avoid generic names such as:

* project
* finalproject
* thesis
* system
* group1
